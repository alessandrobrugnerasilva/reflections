diff
	compara 2 arquivos
		It takes two arguments - the two commit ids to compare. 

clone
	copia um repositorio, incluindo as coisas do git
		it takes one argument - the url of the repository to copy. 

checkout
	seleciona o commit para mudar
		It takes one argument - the commit ID to restore. 

log
	mostra o log de commits
		q   
			sai dos vários logs

ls
	mostra os arquivos do repository
		-a
			inclusive os hidden

status

add

branch

remote
